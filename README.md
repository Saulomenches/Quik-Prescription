
Criei este projeto pessoal com o objetivo de otimizar o dia a dia de profissionais que realizam cálculos frequentes de medicação. A proposta é reduzir a necessidade de recorrer ao celular a todo momento para fazer contas: basta inserir o peso do paciente uma única vez, e a calculadora retorna automaticamente o volume final de forma rápida e precisa.

Além disso, desenvolvi um sistema de prescrição com uso de inputs rápidos, permitindo gerar textos de forma prática, sem a necessidade de digitação extensa. Assim, o processo se torna mais ágil — depois, é só copiar e colar no sistema utilizado para impressão.

O projeto ainda está em desenvolvimento e novas funcionalidades serão adicionadas, mas até aqui tem sido uma experiência muito positiva e produtiva.
